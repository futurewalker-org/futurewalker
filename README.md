# Futurewalker

Modern cross-platform GUI application framework in modern C++.

**Futurewalker is currently in early development phase. Do not expect anything useful at this point.** 

## Project Goals

### Goals

* Supporting wide range of features required by graphics applications.
* Incorporating modern GUI concepts like responsive layouts, gestures, etc.
* Great flexibility, customizability and extensibility.
* Better safety story compared to existing frameworks.
* Better async story compared to existing frameworks.
* Supporting multiple platforms, ideally all of popular ones.
* Good enough energy efficiency and performance.
